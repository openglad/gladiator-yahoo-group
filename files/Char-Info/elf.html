Elf Character

This file contains comments about strategy for use of this character and suggested improvements for this character class.

Strategy

  • Raise intelligent level and experience level until you reach the point where you don't lose mana while walking through a forest.   Attack from within the forest.

Improvements

  • More diversified attacks besides throwing rocks.

  • Also, things need to be reset so that elves can't reach the point of walking through a forest without the loss of mana
