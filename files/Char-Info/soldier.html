Soldier Character

This file contains comments about strategy for use of this character and suggested improvements for this character class.

Strategy

  • Summon a bunch of boomerangs, then go into an area crowded by opponents

Improvements

  • Improve efficiency of mana use for attack

  • Charge could be altered to automatically target an opponent that's in front of the soldier (perhaps in a 10-o'clock-to-2-o'clock range).

  • Place a limit on the number of boomerangs that can be thrown, and add a wider range of spots at which the boomerang will cause damage in its flight.
